- X. Ding, Y. Zhang, T. Liu, J. Duan. Deep Learning for Event-Driven Stock Prediction. IJCAI, 2015
- X. Ding, Y. Zhang, T. Liu, J. Duan. Knowledge-Driven Event Embedding for Stock Prediction. COLING, 2016
